# Lowercase

# lower

Converts a string or content to lowercase.

## Example

```typst
#lower("ABC") \
#lower[*My Text*] \
#lower[already low]
```

```typst
#lower(
  text
) -> str content
```

## Parameters

- text:
  - description: The text to convert to lowercase.
  - type: str | content
  - default: None
