# Binomial

# math.binom

A binomial expression.

## Example

```typst
$ binom(n, k) $
$ binom(n, k_1, k_2, k_3, ..., k_m) $
```

```typst
#math.binom(
  upper,
  lower
) -> content
```

## Parameters

- upper:
  - description: The binomial\'s upper index.
  - type: content
  - default: None
- lower:
  - description: The binomial\'s lower index.
  - type: content
  - default: None
